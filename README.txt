INTRODUCTION
------------

Salesforce Drupal intends to push Drupal created Lead to Salesforce through API.

REQUIREMENTS
------------

No special requirements.

INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module.

CONFIGURATION
-------------

 * After Installation go to url DOMAIN/admin/config/sf_drupal/adminsettings or
   Admin -> Configuration -> System -> SalesForce Drupal Configuration
 * Add required configuration there.
